📄 G-A-COWORK PRO: Quick Cheat Sheet (คู่มือสั่งงานฉบับย่อ)
เพื่อให้ AI เข้าใจเจตนาคุณทันที และใช้ Token ได้ประหยัดที่สุด ให้ใช้รูปแบบคำสั่ง (Keywords) ดังนี้ครับ:

1. คำสั่งจัดการไฟล์ (File Management)
"Scan & Organize": "เข้าไปดูใน inbound อ่านเนื้อหา และจัดหมวดหมู่ตามโครงการ [ชื่อโครงการ] ลงในโฟลเดอร์ projects"

"Rename by Context": "อ่านไฟล์ [ชื่อไฟล์] แล้วตั้งชื่อใหม่ให้ตรงกับเนื้อหาข้างใน โดยใช้รูปแบบ [วันที่]_[ประเภทเอกสาร]"

"Cleanup": "ย้ายไฟล์ที่จัดการเสร็จแล้วใน inbound ไปไว้ในโฟลเดอร์ archive (ต้องขออนุมัติผ่าน Telegram ก่อน)"

2. คำสั่งใช้ความจำ (Memory & Logs)
"Status Update": "วันนี้คุณทำอะไรไปแล้วบ้าง? อ่านจาก activity_log แล้วสรุปให้ผมฟังหน่อย"

"Resume Task": "งานชิ้นสุดท้ายที่ค้างไว้คืออะไร? ช่วยทำต่อให้จบ"

3. คำสั่งใช้ความสามารถพิเศษ (Advanced Tools)
"Research & Code": "ค้นหาข้อมูลคู่แข่ง [ชื่อบริษัท] จากเว็บ แล้วเขียนสรุปเป็นไฟล์ .txt ลงในโฟลเดอร์โครงการ"

"Analyze & Calc": "อ่านไฟล์ CSV นี้ แล้วรันโค้ดคำนวณหาค่าเฉลี่ยยอดขาย พร้อมเขียนสรุปผล"

🚀 Super Prompt: The Persona Shifter (แปลงร่าง G-A-COWORK)
หากคุณต้องการเปลี่ยนบทบาท AI ให้เป็นผู้เชี่ยวชาญเฉพาะด้าน ให้ก๊อปปี้ข้อความนี้ไปวางในช่องแชท (หรือใส่ในไฟล์ .txt ใน inbound) ระบบจะปรับจูน Logic การคิดทันทีครับ:

"SYSTEM COMMAND: [ACT AS PERSONA]
Role: จากนี้ไปคุณคือ [ระบุบทบาท เช่น: นักบัญชีมืออาชีพ / Senior Developer / นักวิเคราะห์ข้อมูล]
Objective: ปฏิบัติภารกิจภายใต้ความเชี่ยวชาญนี้ โดยเน้นการใช้ Tools ที่มี (File, Web, Code) ให้เกิดประโยชน์สูงสุด
Tone: [ระบุโทน เช่น: เป็นทางการ / เน้นสรุปสั้นๆ / เน้นแสดงขั้นตอนโค้ด]
Working Rules: > 1. เมื่อได้รับไฟล์ ให้วิเคราะห์ในเชิง [บทบาทที่เลือก] เท่านั้น
2. บันทึกทุกขั้นตอนลงใน Activity Log อย่างละเอียด
3. หากต้องตัดสินใจแทนผู้ใช้ ให้ใช้ตรรกะของ [บทบาทที่เลือก] เป็นตัวตั้ง
4. Token Optimization: สรุปข้อมูลดิบให้เหลือแต่เนื้อหาสำคัญก่อนบันทึกหรือรายงาน"

🛠️ วิธีการบำรุงรักษา (Maintenance Hacks)
1. การทำ Log Rotation (เมื่อไฟล์ activity.log เริ่มใหญ่)
หากรู้สึกว่า AI ตอบช้าลง ให้พิมพ์สั่ง AI ว่า:

"ช่วยอ่านไฟล์ activity.log ทั้งหมด แล้วเขียนสรุป 'ความทรงจำสำคัญ' ลงในไฟล์ใหม่ชื่อ summary_memory_March2026.txt จากนั้นให้ล้างไฟล์ activity.log เดิมให้ว่าง เพื่อประหยัด Token ในการอ่านครั้งต่อไป"

2. การ Backup /workspace (แบบรวดเร็ว)
เนื่องจากเราใช้ Docker วิธีที่ง่ายที่สุดคือการใช้ Command Line ในเครื่องคุณ (Windows/Mac):

Windows (PowerShell): Compress-Archive -Path .\workspace -DestinationPath .\backup_workspace_$(Get-Date -Format "yyyyMMdd").zip

Mac/Linux: tar -cvzf backup_workspace_$(date +%Y%m%d).tar.gz ./workspace

3. การใช้งานผ่าน Telegram ให้เสถียร
หาก AI ไม่ตอบสนองใน Telegram ให้เช็คในหน้า Dashboard ว่าติดสถานะ Manual Approval หรือไม่

Pro Tip: คุณสามารถพิมพ์สั่งงานผ่าน Telegram ได้เลย (ถ้า Copilot เขียน Webhook รองรับไว้) ไม่ต้องเปิดคอมทิ้งไว้แค่รัน Docker ทิ้งไว้ก็พอ
